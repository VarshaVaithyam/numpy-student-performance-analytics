# 🎓 Student Performance Analytics using NumPy

A beginner-friendly data analysis project built using **Python** and **NumPy**. This project simulates the analysis of student marks by generating random data and performing various statistical and analytical operations.

---

## 📌 Project Overview

This project demonstrates how NumPy can be used to efficiently analyze student performance without using loops. It covers the core NumPy concepts required for data analysis and AI engineering.

The project generates marks for 100 students and performs operations such as:

- Statistical analysis
- Filtering student records
- Sorting marks
- Broadcasting
- Vectorized arithmetic
- Performance reporting

---

## 🚀 Features

- Generate marks for 100 students using NumPy
- Calculate:
  - Average Marks
  - Highest Marks
  - Lowest Marks
  - Median
  - Standard Deviation
- Filter students based on different conditions
- Find:
  - Students scoring above 90
  - Students scoring below 40
  - Students scoring between 60 and 80
- Sort marks in ascending and descending order
- Display Top 10 and Lowest 10 marks
- Perform scalar and vectorized arithmetic operations
- Demonstrate NumPy broadcasting
- Generate a complete student performance report
